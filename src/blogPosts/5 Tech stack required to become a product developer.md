# Tech stack required to become a product developer

Date: 25 Dec 2025  
Place: Chennai

Following are the teck stack one need to master to become a Product developer.  
You need a frontend, backend, both SQL and NoSQL knowledge.

### Front end

- ReactJs, NextJS or Angular

### Back end

- ExpressJS, NodeJS or Python

### Database

- MongoDB, Mongoose, PostgreSQL and Prisma

### Cloud

- AWS or Azure
- Most startups use managed services: Vercel, Render, Fly.io, Netlify

### Infrastructure and tools

- Docker, K8, Jenkins, Git, Postman, Jira
